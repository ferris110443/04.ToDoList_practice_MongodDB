# 安裝

1. 取得專案 clone
`https://github.com/ferris110443/04.ToDoList_practice_MongodDB.git`
   

2. 移到資料夾 install npm
`npm install`


3. mongoDB server starting and initializing 
 ` mongod`

4. modify DBURL to create DB the userDB
`const dbURL = "mongodb://0.0.0.0:27017/userDB";`

5. open localhost
`http://localhost:3000/`
